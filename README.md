# Constraint by Humanoid - Unity Editor Extension

A Unity editor extension that configures IConstraint component based on Unity Humanoid Rules.

## Requirements

- Unity 2018.4.20f1
- Unity 2019.4.31f1

## How to use

None, this editor extension is automatically enabled/worked
