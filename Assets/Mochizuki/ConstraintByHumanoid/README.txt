# Unity Editor Extension - ConstraintByHumanoid

Unity の Humanoid 規格に基づいて、Constraint を自動で設定してくれるエディター拡張です。
元は実験段階のアバターギミックがあったのですが、 VRChat の仕様上不可能だと判明したので、有用だと思った部分のみ切り出して配布しています。
また、 VRChat 向けでの使用を想定していますが、 VRCSDK には依存していないので、 Unity 単体での使用も可能です。

なお、プロジェクト本体は GitHub にて公開されており、知識があれば誰でも無料で使用およびシステムの修正が可能です。
https://github.com/mika-f/Unity-ConstraintByHumanoid


## 注意事項

不具合などがあるかもしれません。ご了承ください。
必ず操作前に Prefab 等の形式でバックアップを作成してから実行してください。
なお、以下の環境にて動作を確認しております。

* Unity 2018.4.20f1


## GitHub 版との違いについて

BOOTH ではビルド済みの UnityPackage と説明書を付けたパッケージを配布しています。
GitHub 版でもほぼ同様のものは作成可能ですが、サポートなどは行いません。
